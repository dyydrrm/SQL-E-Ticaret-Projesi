# E-Ticaret Yönetim Sistemi – SQL Projesi

Bu proje PostgreSQL kullanılarak hazırlanmıştır.  
İçeriklerde;

- Veritabanı tasarımı (7 tablo)
- Örnek veriler (INSERT)
- Fonksiyonlar (calculate_order_total, customer_lifetime_value, stock_status)
- Triggerlar (stok azaltma, fiyat loglama, stok geri yükleme)
- Stored Procedure’ler (sp_place_order, sp_cancel_order)
- View’lar (kategori satış istatistiği, müşteri sipariş özeti)
- Kompleks SQL sorguları

Proje tamamen PL/pgSQL ile geliştirilmiştir.
